# Neo Brutalist Style Reference

PracticeHelper 前端统一使用 `新野兽派（neo-brutalist）` 风格。

## Core Signals

- 粗边框
- 硬阴影
- 无圆角
- 高对比
- 功能主义
- expressive

## Design Rules

- 边框统一使用纯黑色
- 组件默认使用直角，不使用圆角
- 阴影使用硬边缘阴影，不使用模糊阴影
- 主背景保持高对比，强调色只做点缀
- 页面结构要清晰，优先服务信息表达

## Preferred Tokens

- 边框：`border-black border-2 md:border-4`
- 阴影：`shadow-[4px_4px_0px_0px_rgba(0,0,0,1)] md:shadow-[8px_8px_0px_0px_rgba(0,0,0,1)]`
- 圆角：`rounded-none`

## Avoid

- `rounded-lg`
- `rounded-md`
- `rounded-xl`
- `bg-gradient-*`
- `border-gray-*`
- `border-slate-*`
